# Crypto AI Trading Feedback Loop

An autonomous trading agent that fetches real-time crypto data, performs AI-driven price prediction, and calculates risk-managed positions.

## Features
- **Data Scraping:** Automated BTC/ETH data retrieval via Apify.
- **AI Prediction:** Time-series trend analysis using the Kronos foundation model.
- **Risk Management:** Kelly Criterion-based position sizing using market consensus odds.
